3. 生存パターンの概要

全体の生存状況

Target

本データセットに含まれる 714 名のうち、 生存者は約 4 割にとどまります。

0（死亡） 424 59.4%

機械学習タスクとしては、負例（死亡）がやや多い 二値分類問題とみなせます。

性別別の生存率

Key Driver

性別は、生存を説明するうえで最も強い変数の一つです。

4. 連続変数と生存の関係

年齢と生存

Demographic

年齢は、性別やクラスほどではないものの、生存に一定の影響を与えています。

survived 平均年齢 中央値
0（死亡）30.6 歳28 歳
1（生存）28.3 歳28 歳
生存者の方がやや若い傾向がありますが、中央値はいずれも 28 歳であり、 性別やクラスほど明確な差ではありません。 年齢分布を生存有無で重ね描きすると、 子ども（特に低年齢層）の生存率が高いことが視覚的に確認できます。

運賃と生存

Wealth Proxy

運賃は、クラスや客室の質を反映する連続値であり、 生存率との相関がはっきり見られます。

survived 平均運賃 中央値
0（死亡）23.011.89
1（生存）51.826.25
生存者の平均・中央値はいずれも非生存者の 2 倍以上であり、 高額な運賃を支払った乗客（主に上位クラス）が 生存しやすかったことが数値として表れています。

5. まとめと次のステップ

考察の要約

Insights

本レポートの探索的分析から、次のようなパターンが明らかになりました。

  • 生存はランダムではなく、明確な構造を持つ。 特に性別と客室クラスが生存を強く左右しています。
  • 性別とクラスの組み合わせが最も強い要因。 1等・2等の女性は非常に高い生存率を示す一方、 3等の男性は最も厳しい結果となりました。
  • 運賃は富や客室の質の代理変数として機能。 高額運賃の乗客ほど生存率が高く、クラス情報を補完する指標です。
  • 年齢は補助的な説明要因。 生存者はやや若い傾向にあり、子どもが優先された可能性が示唆されますが、 性別・クラスほど強力な決定要因ではありません。
  • 欠損値がないため、前処理の負担が小さい。 この抽出データは教師あり学習や教材用途に非常に適しています。

今後の分析ステップとしては、たとえば次のような方向性が考えられます。

  • 氏名 name から敬称（Mr, Mrs, Miss, Master など）を抽出し、 年齢・性別・社会的地位を反映する特徴量として利用する。
  • sibsp と parch から家族人数 （family_size = sibsp + parch + 1）を作成し、 単身か大家族かといった要素をモデルに組み込む。
  • ロジスティック回帰や決定木系モデルを構築し、 各特徴量の予測力を定量的に評価する。
  • 生存予測モデルが性別・クラスの偏りをどのように再現・強調するか、 公平性の観点から検討する。
